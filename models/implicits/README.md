# Implicit CAD Examples

Browser-native implicit CAD examples for the implicit CAD runtime and CAD
Viewer raymarching/export workflows. Each file is self-contained JavaScript
describing one signed-distance-field model.

## Contents

- `*.implicit.js`: implicit CAD model definitions.

There is no generated catalog sidecar in this committed tree. Viewers and tools
should discover examples from the files in this directory or generate any local
catalog as a transient artifact.
