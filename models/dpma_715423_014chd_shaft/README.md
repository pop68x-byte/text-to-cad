# Вал ДПМА.715423.014ЧД — проверочный CAD-прототип

Сгенерирована упрощённая 3D-модель вала по приложенному чертежу. Модель предназначена только для визуальной и геометрической проверки прототипа, а не для выпуска или производства.

## Файлы в репозитории

В PR хранятся только текстовые файлы:

- `dpma_715423_014chd_shaft.py` — исходный Python-генератор модели.
- `README.md` — описание модели, допущений и способов генерации.

STEP/STL/ZIP не хранятся в репозитории, чтобы PR оставался текстовым и не содержал бинарных CAD-outputs. Эти файлы генерируются локально или через GitHub Actions.

## Генерируемые файлы

При запуске генератора создаются локальные выходные файлы рядом с исходником:

- `dpma_715423_014chd_shaft.step` — нейтральный STEP-файл; экспорт выполнен как фасеточная BREP-геометрия.
- `dpma_715423_014chd_shaft.stl` — STL-файл для быстрого предпросмотра.

Workflow GitHub Actions дополнительно упаковывает эти outputs в:

- `dpma_715423_014chd_shaft.zip`

Эти CAD-outputs игнорируются `.gitignore` и должны скачиваться как artifacts, а не коммититься.

## Основные номинальные размеры

- Общая длина вала: **426 мм**.
- Основная ось детали: **X**.
- Основная цилиндрическая часть: **Ø22 мм**.
- Правый концевой участок: **Ø24 мм**, от правого торца до правого края зоны центрального паза **170 мм**.
- Правый торец: приближённая фаска **2×45°**.
- Левая ступенчатая зона: общая длина **47,5 мм**.
- Первый левый участок: длина **24 мм**.
- Центральный продольный паз:
  - длина **40 мм**;
  - ширина **8 мм**;
  - глубина **4 мм**;
  - радиусные торцы **R4**;
  - расположен на верхней образующей цилиндра.
- Сечение Б-Б / лысковый участок: квадрат/лыски **17 мм** на левом торцевом участке длиной **24 мм**.
- Острые кромки: в исходных параметрах учтена только крупная правая фаска; мелкое притупление 0,2–0,4 мм описано как требование чертежа, но не моделировалось везде детально.

## Допуски и справочная информация с чертежа

Допуски не моделировались геометрически. Для справки:

- Общая длина: **426 ±0,5 мм**.
- Левая ступенчатая зона: **47,5 +0,2 мм**.
- Первый левый участок: **24 +0,2 мм**.
- Паз: **40 ±0,3 мм**, ширина **8 H9**, радиусы в углах **R0,25**.
- Правый участок до паза: **170 ±0,3 мм**.
- Основной диаметр: **Ø22 e8**.
- Правый диаметр: **Ø24 e8**.
- Квадрат/лыски: **17 -0,1 мм**.
- Острые кромки притупить радиусом или фаской **0,2…0,4 мм**.
- Шероховатости **Ra1,6** и **Ra3,2** не моделировались.

## Материал

Материал указан справочно по требованиям чертежа:

- σ0,2 ≥ **635 МПа**;
- σв ≥ **830 МПа**;
- KCV-40 ≥ **25 Дж/см²**.

Допускается замена материала на **20Х13**, **14Х17Н2**, **07Х16Н6**, **07Х16Н4** по **ГОСТ 5632-2014**.

## Принятые допущения

- Левая зона выполнена упрощённо: участок 0–24 мм сделан как квадратный/лысковый участок 17 мм в габарите Ø24, участок 24–47,5 мм — как Ø24, далее переход к Ø22.
- Точная форма левого торцевого уступа и мелкой проточки у самого торца на чертеже неоднозначна; она не детализировалась сверх сохранения длины зоны и номинальных диаметров 22/24 мм.
- Паз смоделирован как продольная фрезерованная выборка на верхней образующей с номинальной шириной 8 мм, глубиной 4 мм и торцами R4; радиусы внутренних углов R0,25 в фасеточной модели явно не выделялись.
- STEP создаётся как фасеточная BREP-модель из той же сетки, что и STL. Это нейтральный проверочный STEP, но не параметрическая производственная твердотельная модель.
- Мелкие притупления 0,2–0,4 мм и шероховатости не воспроизводились детально.
- Модель не подтверждает полное соответствие чертежу и не должна использоваться как финальная производственная документация.

## Локальная генерация

Из корня репозитория выполните:

```bash
python models/dpma_715423_014chd_shaft/dpma_715423_014chd_shaft.py
```

После запуска будут созданы или перезаписаны локальные файлы:

- `models/dpma_715423_014chd_shaft/dpma_715423_014chd_shaft.step`
- `models/dpma_715423_014chd_shaft/dpma_715423_014chd_shaft.stl`

Эти файлы намеренно не коммитятся.

## Генерация через GitHub Actions

1. Откройте вкладку **Actions** в GitHub-репозитории.
2. Выберите workflow **Generate DPMA shaft CAD outputs**.
3. Нажмите **Run workflow** и запустите его для нужной ветки.
4. После завершения run откройте страницу этого run.
5. В блоке **Artifacts** скачайте artifact `dpma-715423-014chd-shaft-cad-outputs`.

В artifact будут находиться сгенерированные STEP, STL и ZIP.

## Путь к STEP после генерации

Локально или внутри скачанного artifact STEP находится по пути:

`models/dpma_715423_014chd_shaft/dpma_715423_014chd_shaft.step`
